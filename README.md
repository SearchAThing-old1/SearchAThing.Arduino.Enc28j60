# SearchAThing.Arduino.Enc28j60
Enc28j60 driver for Arduino SearchAThing network subsystem.
- Basic ( static ip )
- Basic ( dynamic ip )
- SRUDP ( echo client )
- SRUDP ( echo server )
- SRUDP-TCP-Bridge
- Arduino Servo

Examples available [here](https://github.com/devel0/SearchAThing.Arduino.Enc28j60.Examples)
